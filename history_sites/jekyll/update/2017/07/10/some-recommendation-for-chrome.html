Chrome Plugin Recommendation

Jul 10, 2017

A few good stuff I scavenged from Chrome wasteland.

My Chrome Plugins List

  • Stylish, you has to get it if you like to change e.g. Google search page, to dark theme. It’s better to use it together with this chrome colorscheme.
  • Surfingkey, you cannot miss if you are looking for a really good vim key-binding plugin in Chrome. Check my config files if you are interested!
  • Wasavi, when surfing key plugin cannot be used for some text intput box
  • Emoji Helper, nice tool to pick your emoji and copy to clipboard
  • FoxClocks, useful if you work across timezone. Don’t forget to set the status bar option to “use ‘transform’” to avoid breaking website.
  • Display Anchor, specially handy if you want copy the “jump link” (e.g. jump to a particular answer in StackOverflow’s page)
  • uSelect iDownload, multiple link select and download
  • Clear Cache, Dev Tool: clear not just cache, but also others like cookies, localStorage, indexedDB etc.
  • JSON view, Dev Tool: json viewer instead of plain text.
  • Web Maker, the offline Codepen for FrontEnd.
  • Wappalyzer, Dev Tool: peek certain technologies used in current website.
  • TamperMonkey, the radiated monkey living in chrome wasteland (similar species to greasemonkey), with them you could do e.g. youtube mp3 download button and … . But watch out malicious scripts!
  • Send to Kindle, where most TLDR goes
  • Google Mail Checker, handy if you like web gmail client and have shortcut keybinding e.g. cmd+G
  • Clean Youtube, ads/recommendation free (less distraction).
  • Screen Capture for the entire webpage and export to pdf when default printer doesn’t work!
