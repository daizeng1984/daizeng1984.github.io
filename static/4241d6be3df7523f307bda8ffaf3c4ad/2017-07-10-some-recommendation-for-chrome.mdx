A few good stuff I scavenged from Chrome wasteland.

## My Chrome Plugins List

* [Stylish](https://userstyles.org/), you has to get it if you like to change e.g. Google search page, to [dark theme](https://userstyles.org/styles/118959/darksearch-for-google). It's better to use it together with [this](https://chrome.google.com/webstore/detail/material-dark/npadhaijchjemiifipabpmeebeelbmpd) chrome colorscheme.
* [Surfingkey](https://github.com/brookhong/Surfingkeys/issues/31), you cannot miss if you are looking for a really good vim key-binding plugin in Chrome. Check my [config](https://github.com/daizeng1984/dotfiles/blob/master/misc/surfkeys.js) files if you are interested!
* [Wasavi](https://github.com/akahuku/wasavi), when surfing key plugin cannot be used for some text intput box
* [Emoji Helper](https://chrome.google.com/webstore/detail/emoji-cheatsheet-for-gith/nojknakoailnpbjlmfkpbbhoodlolfbh?hl=en), nice tool to pick your emoji and copy to clipboard
* [FoxClocks](https://chrome.google.com/webstore/detail/foxclocks/obcbigljfpgappaaofailjjoabiikckk/related?hl=en), useful if you work across timezone. Don't forget to set the status bar option to "use 'transform'" to avoid breaking website.
* [Display Anchor](https://chrome.google.com/webstore/detail/display-anchors/poahndpaaanbpbeafbkploiobpiiieko?hl=en), specially handy if you want copy the "jump link" (e.g. jump to a particular answer in StackOverflow's page)
* [uSelect iDownload](https://chrome.google.com/webstore/detail/uselect-idownload/ileabdhfjmgaognikmjgmhhkjffggejc?hl=en), multiple link select and download
* [Clear Cache](https://chrome.google.com/webstore/detail/clear-cache/cppjkneekbjaeellbfkmgnhonkkjfpdn/related?hl=en), Dev Tool: clear not just cache, but also others like cookies, localStorage, indexedDB etc.
* [JSON view](https://chrome.google.com/webstore/detail/jsonview/chklaanhfefbnpoihckbnefhakgolnmc/related?hl=en), Dev Tool: json viewer instead of plain text.
* [Web Maker](https://webmakerapp.com/), the offline Codepen for FrontEnd.
* [Wappalyzer](https://chrome.google.com/webstore/detail/wappalyzer/gppongmhjkpfnbhagpmjfkannfbllamg?hl=en), Dev Tool: peek certain technologies used in current website.
* [TamperMonkey](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo?hl=en), the radiated monkey living in chrome wasteland (similar species to [greasemonkey](https://addons.mozilla.org/en-US/firefox/addon/greasemonkey/)), with them you could do e.g. youtube [mp3 download button](https://greasyfork.org/en/scripts/20015-simple-youtube-mp3-button) and ... ü§ê. **But watch out malicious scripts!**
* [Send to Kindle](https://www.amazon.com/gp/sendtokindle/chrome), where most TLDR goes
* [Google Mail Checker](https://chrome.google.com/webstore/detail/google-mail-checker/mihcahmgecmbnbcchbopgniflfhgnkff?hl=en), handy if you like web gmail client and have shortcut keybinding e.g. <kbd>cmd</kbd>+<kbd>G</kbd>
* [Clean Youtube](https://chrome.google.com/webstore/detail/remove-youtube-recommende/hfnghnjkcpgagjbiheidccjnneoipbmb?hl=en), ads/recommendation free (less distraction).
* [Screen Capture](https://chrome.google.com/webstore/detail/take-webpage-screenshots/mcbpblocgmgfnpjjppndjkmgjaogfceg?hl=en) for the entire webpage and export to pdf when default printer doesn't work!
