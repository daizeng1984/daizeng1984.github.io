# Background Setup
We need to set the background to universe. Once you set it as universe, it brings so much flexibility to connect across different titles and make your story logical and convincing when you bring all heroes & heorines together.

We need a big villain and a mainline story. Let's create it: `Thinos` mapping to Marvel's `Thanos`.
